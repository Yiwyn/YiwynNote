#### <font color='red'>使用@代替根目录</font>



> ##### 环境 vue3 vite  typescript



##### tsconfig.json

```typescript
{
	"compilerOptions": {
		...
    	"baseUrl": "./",
	    "paths": {
    	  "@/*": ["src/*"]
	    }
	}
}
```



##### vite.config.ts

##### 首先引入path (npm i path)

```typescript
import path from 'path'

// https://vitejs.dev/config/
export default defineConfig({
  plugins: [
    vue({
      template: { transformAssetUrls },
    }),
  ],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, 'src'),
    },
  },
})
```
