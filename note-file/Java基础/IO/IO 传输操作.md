#### IO理解分类-从传输方式上

- ##### 字节流

- ##### 字符流

> ##### ==字节==是给计算机看的，==字符==是给人看的

#### 字节流和字符流的区别

- ##### 字节流读取单个字节，字符流读取单个字符（一个字符根据编码的不同，对应的字节也不同）

- ##### 字节流用来处理二进制文件（MP3、图片、视频文件），字符流用来处理文本文件。
